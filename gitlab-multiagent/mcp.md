Terraform MCP:

The Terraform MCP provides access to the latest Terraform Registry documentation, modules, providers, policies, and version information. Use it to discover Terraform best practices, validate resource configurations, retrieve module documentation, identify provider capabilities, and generate accurate Terraform code based on current registry documentation.


GitLab MCP:

Provides tools to interact with GitLab. Use it to manage repositories, branches, merge requests, and CI/CD pipelines through natural language.


AWS MCP:

Provides read-only access to AWS resources. Use it to inspect existing infrastructure — check VPCs, subnets, S3 buckets, EC2 instances, IAM roles, and other resources. Cannot create, modify, or delete anything. Use it to verify resources after deployment.
